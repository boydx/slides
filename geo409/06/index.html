STEP 1

Make a map

  • Layers should include arches
  • streams, waterbodies, rivers
  • labeled streams and arches
  • and a hillshade background with semi-transparent landuse layer.
  • Export as GeoPDF.

STEP 2

Publish map

  • Create new repo on Github.com
  • Enable GitHub pages
  • Add boydx as a collaborator
  • Clone down new repo, add map, commit, and push
  • Find URL to map, e.g. https://username.github.io/rrg/basemap/rrg-arches.pdf

STEP 3

Load app and data

  • Load Avenza Maps on your mobile device.
    • Basic intro to app
    • Documentation
  • Load RRG base map
    • Use URL of GeoPDF in your repo
    • to add a new map to Avenza Maps.
    • E.g., https://username.github.io/rrg/basemap/rrg.pdf
